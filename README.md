Court Vision is a multi-page Streamlit dashboard that pulls live WNBA player statistics from the official stats API across 7 seasons. It lets you explore per-game stats and advanced efficiency metrics — including True Shooting %, AST/TOV ratio, and per-minute rates — with interactive filters for season, team, and minutes played. Features include a player leaderboard, side-by-side player comparison with career trend charts, and league-wide year-over-year analysis.

## Tech Stack
- Python, Pandas, NumPy
- Streamlit (multi-page)
- Plotly Express
- nba_api
